## Brand at a glance

- **Style:** Minimal, clean, simple
- **Primary palette:** Sage Green (#A1AB97), Off-White (#EDEDED), Cream (#ECDFC9)
- **Key rule:** Never use the word "Natural" in copy — use plant-derived / mineral-based language instead
